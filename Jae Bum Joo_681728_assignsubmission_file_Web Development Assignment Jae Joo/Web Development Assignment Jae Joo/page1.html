So, HOW?

5 ways to be Eco-Friendly and REDUCE waste

  1. Try to use reusable bags instead of plastic ones.
  2. Don't throw it away! Try to Fix it. Don't buy it, Borrow it.
  3. Use Reusable cups instead of paper ones, and try not to use straws.
  4. Use Cloth napkins.
  5. Go Paperless, get bills and letters e-mailed.
