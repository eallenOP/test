How to Help

One of the easiest ways to help the enviroment is to make sure you recycle.

Make sure to only put items listed as allowed in each bin.

Another way one could help is by donating usable items you no longer need or want to Rummage

Rummage accepts:

  • Clothing
  • Books
  • Toys
  • Shoes
  • Knick-knacks
  • Crockery
  • Kitchenware
  • Furniture
  • Beds
  • Tools
  • Electrical items

This would assure said items aren't needlessly thrown out and adding to the landfills.
